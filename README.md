# To use this library
<b>Step 1. Add it in your root build.gradle at the end of repositories:</b>
<br>
<p>
    allprojects {<br>
         repositories {<br>
           ...<br>
           maven { url 'https://www.jitpack.io' }<br>
        }<br>
    }<br>
</p>
<p>
<b>Step 2. Add the dependency:</b><br>
dependencies {<br>  
    implementation 'com.github.deeqxaajidaahir:TaatikoPlaces:Tag'
    <br>
}
</p>
